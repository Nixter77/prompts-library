## 2025-05-18 - [Manual Sizing for Lucide Loader]
**Learning:** The `Loader2` component from `lucide-react` does not have default sizing suitable for button icons and requires explicit classes (e.g., `h-4 w-4 mr-2`) to align correctly with text.
**Action:** Always apply `h-4 w-4` and appropriate margin when using `Loader2` inside a Button.
