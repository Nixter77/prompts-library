# Palette Journal

## 2026-01-24 - Form Accessibility & Feedback Pattern
**Learning:** Forms initially used `window.alert()` for error feedback and lacked explicit `<label>` elements (relying on placeholders), creating an inaccessible and disruptive user experience.
**Action:** Replace all `alert()` usage with inline error messages (e.g., red text with icons) and ensure every input has a visible, associated `<label>`.
