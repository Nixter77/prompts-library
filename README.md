# prompts-library

Простой каталог и библиотека текстовых промптов для использования в проектах и для демонстрации.

Это Next.js приложение с локальной базой данных SQLite, в котором можно просматривать, добавлять и фильтровать промпты по категориям и тегам.

## Что внутри

- Интерфейс на Next.js (App Router).
- Лёгкая локальная база данных — `SQLite` (файл `database.sqlite`).
- API-роуты в `src/app/api/prompts` для получения/создания/удаления промптов.

## Технологии

- Next.js 15
- React 19
- Tailwind CSS
- SQLite + `sqlite3` (локальная база)

## Быстрый старт (локально)

Требования:

- Node.js (рекомендуется 18+)
- npm или yarn

Установка зависимостей:

```bash
npm install
# или
yarn
```

Запуск в режиме разработки:

```bash
npm run dev
```

Инициализация базы данных (создаст `database.sqlite` и таблицу `prompts`):

```bash
node src/db/init.js
```

Сборка для продакшна и запуск:

```bash
npm run build
npm run start
```

Проверка lint (ESLint):

```bash
npm run lint
```

## Структура проекта (коротко)

- `src/app` — страницы приложения (App Router).
	- `/add-prompt` — страница добавления промпта.
	- `/prompts/[id]` — просмотр промпта.
	- `/prompts/categories/[category]` — фильтр по категории.
- `src/app/api/prompts` — API-роуты для CRUD-операций над промптами.
- `src/db/init.js` — скрипт инициализации базы данных.
- `database.sqlite` — файл базы данных (создаётся локально).

## Как использовать

1. Запустите приложение локально (`npm run dev`).
2. Откройте в браузере http://localhost:3000
3. Перейдите на страницу «Добавить промпт» (`/add-prompt`) для создания нового промпта.
4. Просматривайте промпты в категориях или индивидуально по id.

API:

- GET `/api/prompts` — получить список промптов.
- GET `/api/prompts/:id` — получить промпт по ID.
- POST `/api/prompts` — создать новый промпт.
- DELETE `/api/prompts/:id` — удалить промпт.

(См. реализацию в `src/app/api/prompts`.)
